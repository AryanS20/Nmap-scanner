# Nmap-scanner
This project is a cybersecurity-focused Python application that automates basic network reconnaissance tasks using three widely used tools: Ping, ARP, and Nmap.

The goal is to build multiple Python scripts that interact with system-level networking commands using the subprocess module, process their outputs, and present meaningful results to the user.

In simple terms, you are creating a mini network scanning toolkit that helps identify active devices, map IP-to-MAC addresses, and scan open ports or services on a network
